# Prompt Engineering Projects 🤖

A collection of 3 beginner-friendly AI projects built using Python and the OpenAI API, exploring the basics of prompt engineering.

## Projects Included

1. 😄 Joke Generator — Give a topic, get a funny joke
2. 💬 Text to Emoji — Converts any sentence into emojis
3. 🎯 Interview Question Generator — Generate interview questions on any topic

## Tech Used
- Python
- OpenAI API
- Prompt Engineering
